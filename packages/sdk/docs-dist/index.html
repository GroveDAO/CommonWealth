@commonwealth/sdk v1.0.0

CommonWealth SDK

Typed contract clients, ABI exports, and storage helpers for the CommonWealth treasury stack. The SDK is designed to support live Sepolia governance flows, Zama private voting integrations, Storacha upload tooling, and DePIN reward automation.

Install

pnpm add @commonwealth/sdk viem wagmi

Primary Exports

  • ABI constants for all deployed contracts
  • Typed clients for governance, impact, savings, and DePIN
  • Storage integration via w3up-compatible clients
  • Enum exports for circle state and data type routing

Usage

import { ConvictionVotingClient } from "@commonwealth/sdk";

const client = new ConvictionVotingClient({
  address: "0xCe7b7301b29CC8D00a508a2900dc5D5B900176Af",
  chainId: 11155111,
});
